\# Curriculum Market Intelligence Engine (CMIE)



AI-driven curriculum product generation system for creating sellable teaching resources.



\## Main features



\- Lesson generation

\- PowerPoint slide generation

\- Presenter notes generation

\- Assessment generation

\- Workbook generation

\- Packaging and release pipeline
